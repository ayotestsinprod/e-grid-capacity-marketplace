# Mapbox Token Setup

To complete the deployment:

1. Go to https://account.mapbox.com/auth/signup/
2. Sign up with: aoluwanusin@gmail.com
3. Create a new access token with default public scopes
4. Add to Vercel environment variables as NEXT_PUBLIC_MAPBOX_TOKEN

Or use existing account at https://account.mapbox.com/access-tokens/
